命令行格式约定

格式

意义

粗体

命令行关键字（命令中保持不变、必须照输的部分）采用加粗字体表示。

斜体

命令行参数（命令中必须由实际值进行替代的部分）采用斜体表示。

[ ]

表示用“[ ]”括起来的部分在命令配置时是可选的。

{ x | y | ... }

表示从两个或多个选项中选取一个。

[ x | y | ... ]

表示从两个或多个选项中选取一个或者不选。

{ x | y | ... } *

表示从两个或多个选项中选取多个，最少选取一个，最多选取所有选项。

[ x | y | ... ] *

表示从两个或多个选项中选取多个或者不选。

图形界面元素引用约定

格式

意义

“”

带双引号“”的格式表示各类界面控件名称和数据表，如单击“确定”。

>

多级菜单用“>”隔开。如选择“文件 > 新建 > 文件夹”。

键盘操作约定

格式

意义

加“”的字符

表示键名。如“Enter”、“Tab”等分别表示回车、制表。

“键1+键2”

表示在键盘上同时按下几个键。如“Ctrl+Alt+A”表示这三个键同时按下。

“键1，键2”

表示先按第一键，释放，再按第二键。如“Alt，F”表示按先后顺序按键。

鼠标操作约定

格式

意义

单击

快速按下并释放鼠标的一个按钮。

双击

连续两次快速按下并释放鼠标的一个按钮。

拖动

按住鼠标的一个按钮不放，移动鼠标。

文档操作说明

操作

意义

单击图片

弹出图片独立浏览窗口。当默认浏览模式下无法看清图片时，请用图片独立浏览窗口查看。

按“Alt+←”键

返回上一次浏览的页面。

按“Alt+→”键

撤销返回操作。
