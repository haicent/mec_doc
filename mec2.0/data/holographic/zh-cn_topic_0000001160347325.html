用于登录Web页面。

首次登录时，请按照提示设置密码。

首次登录会提醒设置密码。

admin

SDC 8.2.0版本之前：HuaWei123；

SDC 8.2.0版本及之后：无默认密码。

用途：

  • 用于使用SDK协议对接平台。
  • 用于应急维护设备认证，通过iClient SDC软件实现对摄像机进行重启、升级等应急维护操作。
  • 用于复眼型摄像机动点和定点的交互认证，使用的是动点的该账号和密码。

SDK协议与应急维护设备认证使用的同一个密码，密码修改的具体步骤如下：

  1. 使用管理员帐户登录摄像机的Web界面（https://IP地址）。
  2. 选择“配置 > 网络 > 平台对接参数 > 密码管理”。
  3. 在“SDK密码管理”区域输入“Web用户密码”、“新密码”和“确认密码”。
  4. 单击“保存”，完成密码的修改。

admin

SDC 8.2.0版本之前：HuaWei123；

SDC 8.2.0版本及之后：无默认密码。

用于使用第二协议（ONVIF）对接平台及使用RTSP协议请求实况。

密码修改的具体步骤如下：

方法一：

  1. 使用管理员帐户登录摄像机的Web界面（https://IP地址），单击“设置”。
  2. 选择“配置 > 网络 > 平台对接参数 > 密码管理”。
  3. 在“ONVIF密码”区域输入“Web用户密码”、“新密码”和“确认密码”。
  4. 单击“保存”，完成密码的修改。

方法二：

  1. 使用管理员帐户登录摄像机的Web界面（https://IP地址），单击“设置”。
  2. 选择“配置 > 系统配置 > 密码管理”。
  3. 选择“协议密码”。
  4. 选择“ONVIF密码”，输入“Web用户密码”、“新密码”和“确认密码”。
  5. 单击“保存”。

ApiAdmin

SDC 8.2.0版本之前：HuaWei123；

SDC 8.2.0版本及之后：无默认密码。

用途如下：

  • 用于使用RESTful协议对接摄像机。
  • 用于使用SDK协议对接摄像机。
  • 用于应急维护设备认证，通过iClient SDC软件实现对摄像机进行重启、升级等应急维护操作。
  • 用于铁路周界场景下雷达对接摄像机。
  • 用于生态型摄像机的子母机对接。
  • 用于生态型摄像机的子机与微波的RT板模块对接。
说明：
  • ApiAdmin账号用于SDK协议对接摄像机时，需要客户端支持，部分老平台无法使用。
  • 用于SDK协议对接或者应急维护设备认证时，ApiAdmin账号与admin账号权限相同。

密码修改的具体步骤如下：

  1. 使用管理员帐户登录摄像机的Web界面（https://IP地址）。
  2. 选择“配置 > 网络 > 平台对接参数 > 密码管理”。
  3. 在“北向接口密码”区域输入“Web用户密码”、“新密码”和“确认密码”。
  4. 单击“保存”，完成密码的修改。

root

HuaWei123

Linux系统的管理员帐户。

方法一：

  • 使用命令passwd user_name修改密码。例如使用passwd admin命令修改admin用户的密码。
  • root用户无法直接登录Linux系统。在admin用户登录后，可以通过命令su - root切换为root用户。

方法二：

  1. 选择“配置 > 网络 > SSH/SFTP > Linux密码”。
  2. 选择“用户名”，可选择“admin”、“root”。
  3. 输入“Linux用户密码”、“新密码”及“确认密码”。
  4. 单击“保存”。

方法三：

  1. 选择“配置 > 系统配置 > 密码管理 > Linux密码”。
  2. 选择“用户名”，可选择“admin”、“root”。
  3. 输入“Linux用户密码”、“新密码”及“确认密码”。
  4. 单击“保存”。

sshd

-

为了提高系统安全性，运行进程时建议使用该用户，避免使用root用户。

该帐户无法登录，不涉及到密码。

TF卡PIN码

HuaWei123

该密码对应TF卡的PIN码和TF卡的管理PIN码。

用于SDC和TF卡进行鉴权认证。

目前使用的TF卡为三未信安的TF卡。

密码修改的具体步骤如下：

  1. 使用管理员帐户登录摄像机的Web界面（https://IP地址）。
  2. 选择“配置 > 网络 > 平台对接参数 > 第二协议参数 > T28181 > GB35114”，勾选“GB35114”。
  3. 单击“修改PIN码”，输入“当前TF卡PIN码”、“新TF卡PIN码”和“确认TF卡PIN码”。
  4. 单击“确定”，完成密码的修改。

该帐户无法登录，不涉及到密码。

ApiAdmin

无

内置RT的插件管理帐号，首次进入“内置RT”页签时，系统会提示要设置ApiAdmin用户的密码。该用户名和密码用于子机和内置RT内部通信时的鉴权。请根据界面提示设置。
  1. 选择“配置 > 网络 > 无线模块”。
  2. 选择“内置RT”页签，单击“无线模块密码配置”。
  3. 修改无线模块密码。
  4. 单击“保存”。

ApiAdmin

无

外置RT的插件管理帐号，首次进入“外置RT”页签时，系统会提示要设置ApiAdmin用户的密码。该用户名和密码用于子机和外置RT之间通信时的鉴权。请根据界面提示设置。
  1. 选择“配置 > 网络 > 无线模块”。
  2. 选择“外置RT”页签，单击“无线模块密码配置”。
  3. 修改无线模块密码。
  4. 单击“保存”。
